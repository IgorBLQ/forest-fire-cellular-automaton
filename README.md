# Forest Fire Cellular Automaton

This repository contains the implementation of a probabilistic 2D cellular automaton model for wildfire propagation including:

- Local humidity field
- Directional wind vector influence
- Probabilistic ignition rules
- Quantitative CSV export
- Comparative graphical analysis

## How to run

python simulation.py
python analysis.py
